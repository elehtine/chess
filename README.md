# Chess

## Execute
------------------
for now builds from shell "script" in src
------------------
First time:
```
mkdir build && cd build
cmake ..
cmake --build .
./Chess
```
After first time in build directory:
```
cmake --build .
./Chess
```

## Unit tests

In the build directory:
```
cmake --build .
ctest
```
